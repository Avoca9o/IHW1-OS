2. Условие задачи: Разработать программу, которая меняет на обратный 
порядок следования символов каждого слова в ASCII-строке символов. 
Порядок слов остается неизменным. Слова состоят только из букв.
Разделителями слов являются все прочие символы.

3. Cхема решения задачи следующая: она получает на вход имена 
входного и выходного файла через аргументы командной строки. 
Далее создается первый дочерний поток, который порождает первый 
неименованный канал и собственно записывает туда текст из входного файла.
После этого первый дочерний поток закрывается, создается следующий, 
порождается второй неименованный канал. Этот поток считывает данные из первого
неименованного канала, проводит необходимые изменения по заданию и записывает 
новый полученный текст во второй неименованный канал. Первый нам больше не нужен,
поэтому он закрывается. Второй дочерний поток также закрывается. Создается третий,
который считывает весь обработанный текст из второго неименованного канала (текст,
который положил туда второй поток) и записывает его в файл. В конце все 
неименованные каналы закрываются.

4. Действительно разработано консольное приложение, выполняющее эти три треобвания:
- первый процесс читает данные из файла и через неименованный канал передает их 
второму процессу
- второй процесс осуществляет обработку и передает новый текст третьему процессу 
через неименованный канал
- третий процесс осуществляет вывод текста в выходной файл

5. Имена входного и выходного файлов действительно передаются через аргументы
командной строки

6. Ввод и вывод действительно осуществляется при помощи команд read и write

7. Весь текст запиывается в буфер за одно считывание и не превышает 5001 байт

8. Предоставлен набор тестовых файлов

9. Для каждого входного тестового файла предоставлен выходной
(input<i>.txt соответствует output<i>.txt)
