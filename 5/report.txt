1. Схема решения следующая: создаются именованные каналы 
с именами fifo1 и fifo2. Первый процесс считывает данные из 
входного файла и записывает их в fifo1. Второй процесс
считывает данные из fifo1, обрабатывает их как того требует задача
и записывает в fifo2. Третий процесс после этого считывает обработанные
данные из fifo2 и записыват в выходной файл. Имена каналов указаны,
создаются они командой mkfifo(name, flags); открываются при помощи
команды open

2. Действительно разработано консольное приложение, соответствующее 
слуедующим требованиям: 
- первый процесс читает данные из текстового файла и передает их
через именованный канал второму процессу
- второй процесс обрабатывает полученные данные и передает их через
именованный канал третьему процессу
- третий процесс осуществляет вывод данных в текстовый файл

3. Действительно для задания имен входного и выходного файла
используются аргументы командной строки

4. Ввод и вывод осуществляется при помощи команд read и write

5. Данные заносятся за одно считывание и не превышают по размеру 5001 байт

6. Представлен набор тестовых файлов (в папочке tests)

7. Для каждого тестового файла предоставлен файл с результатом выполнения
программы на этом тесте (файл input<i>.txt соответствует output<i>.txt)
